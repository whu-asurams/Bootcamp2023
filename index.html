Introduction

  • Camp name: ASU NASA STEM Summer Boot Camp for HTML+CSS with Github co-pilot (a chatGPT3 based AI)
  • Dates: June 5- June 30, 2023
  • Deadline to Register: June 2
  • Cost: $0
  • Age Requirement: finishing 5th grade to finishing 12th grade.
  • How to Register: Registration form
  • About the Camp: ASU's Center for Blockchain Innovation and Emerging Technology also hosts the first summer boot camp in Web Design. Students will learn how to create webpages using HTML, CSS and Bootstrap framework. We will use the Visual Studio for Code environment, which is further equipped with Github co-pilot, a chatGPT3 based AI extension. The camp starts on Monday, 6/5/2023, and lasts 4 four weeks to end on 6/30/2023, five days a week. We start in the morning at 9:00am and end at 4:00pm, with a lunch break during 12:00pm-1:00pm. After the camp, students will develop a website of four webpages that feature a particular topics of individual choice. Further instructions will be provided to help students create a public website with their own domain names. In particular, students can use this opportunity to develop a website that can be used for college applications and future job applications.

Setup VSC with Github co-pilot
We will use Github co-pilot for our coding. It is an application that is based on chatGPT3. It is a very powerful, and it provide intelligent suggestions of coding or coding blcok, which makes coding much easier. To use Github co-pilot, you need to create a Github account. Please first make sure that you have a valid email account, then use that email account to create a Githun account.

  • To create a valid email account, you can go to gmail to sign up an account.
  • To create a Githun account, please go to Github Signup to create a Github account.
